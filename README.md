# Library Management System (Java)

This project was developed for the Object Oriented Programming 1 course.

## Description
A console-based Library Management System that allows users to:

- Sign up and login
- Browse books
- Borrow books
- Buy books
- Return books
- View most borrowed or bought books

Admins can:
- Add books
- Remove books
- Update book information

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- ArrayList
- HashMap

## OOP Concepts Used
- Inheritance
- Polymorphism
- Encapsulation
- Class hierarchy
